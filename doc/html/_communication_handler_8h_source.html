4 /*
5 This library handles PC to receiver communications
6 Protocol specifications :
7  Without this protocol, the transmitter broadcasts commands to every receiver
8  The transmitter can send commands to one specific receiver
9  Each receiver has an address defined by deviceAddress in class declaration
10  The maximum number of devices is defined by CH_MAX_DEVICE_ADDRESS_LEN
11  Each receiver can send messages / commands to the transmitter
12  by using a letter as the device address, this letter is 'm'
13 
14  This is what a command from transmitter looks like :
15  "m/56:do something\r"
16  m -> sender
17  56 -> receiver
18  do something -> command destined to the specific device
19  '/' -> means "sends to"
20  ':' -> after that character is the command
21  \n -> endline character
22  So any command must look like this :
23  <sender>/<receiver>:<command>\n
24  What if there is only one device on the network ?
25  Syntax must be respected, otherwise, command will be ignored
26 
27  A message from any receiver to the transmitter must be like this :
28  "23/m:i cannot do something\r"
29  All devices will receive the message but will ignore it
30  NOTE : receivers are able to receive messages only from 'm'
31  NOTE : receiver address 'a' means all ; so 'm/a:selftest' will be sent
32  to all receivers
33  NOTE : this protocol is handled by Message.h
34  */
